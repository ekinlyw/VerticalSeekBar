Android VerticalSeekBar
===============

Usage
===============
1.In layout XML,
	
    <com.ekinlyw.widget.VerticalSeekBar
        android:layout_width="wrap_content"
        android:layout_height="200dp"
        android:progress="50" />

2.In the source code,

	VerticalSeekBar vSeekBar = new VerticalSeekBar(Context);
